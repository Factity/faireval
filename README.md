# faireval       
The following repository contains research analaysis for evaluation and mitigating discrimination in language models. Here we xplore different eda methods and test case scenarions to clearly study the bias in ai models. The identified criterias are asssesed studied and the effects of judgment on different demographic profiles in analyzied the results of thye works are present in the following github repository 

# Project scope 

The project is for anyone who is trying to build and deploy ai models into front facing environments where having throughly build safeguards is important. When passing judgement over mission critical scenarios all forms of bias and unfairness hsould be removed. We as a team intend to make ai more helpful, humane and harmless.

## Color Reference for development guide 
# Violet Brand Identity - Color Scheme

## Tool Compatibility
This scheme is optimized for **Tailwind CSS**, **Figma**, and **Sass**.

## Color Palette


| Color Name      | Hex Code | Role |
| :---           | :---     | :--- |
| **Deep Violet**  | `#330067` | Primary Background / Headings |
| **Amethyst**     | `#775BBE` | Primary Brand Color / Buttons |
| **African Violet**| `#B284BE` | Secondary / Accents / Hover States |
| **Electric Yellow**| `#FFD700` | Call to Action (CTA) / Highlights |
| **Soft Cream**    | `#F9F6EE` | Body Text / Neutral Background |

## UI Element Mapping

### 1. Typography
*   **Headings (h1, h2, h3):** `Deep Violet (#330067)` - Provides high contrast and authority.
*   **Body Text:** `Deep Violet` (at 90% opacity) on `Soft Cream`.

### 2. Layout & Surfaces
*   **Primary Background:** `Soft Cream (#F9F6EE)` - Keeps the interface readable.
*   **Header/Footer:** `Deep Violet (#330067)` - Frames the content strongly.
*   **Cards/Containers:** White with a subtle border in `African Violet (#B284BE)`.

### 3. Interactive Elements
*   **Buttons (Primary):** `Amethyst (#775BBE)` with White text.
*   **Buttons (Hover):** `African Violet (#B284BE)`.
*   **CTA Button (Urgent):** `Electric Yellow (#FFD700)` with `Deep Violet` text to grab attention.

### 4. Feedback & Accents
*   **Active States/Links:** `Amethyst (#775BBE)`.
*   **Dividers/Borders:** `African Violet (#B284BE)` at low opacity.

## Run Locally

Clone the project

```bash
  git clone https://github.com/Factity/faireval.git
```

Go to the project directory

```bash
  cd faireval
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

## Python server

It is reccomended to use conda or create a virtual environment 

```bash
conda create -n YOUR_VIRTUAL_ENV_NAME python=3.12
```
Activate your virtual environment 

```bash
conda activate YOUR_VIRTUAL_ENV_NAME 
```

Install the required requirements 

```bash
pip install -r requirements.txt
```

Run the flask app locally using the command 

```bash 
python app.py 
```
